A source of ZIP Code™ information.

Usage

Go to

/api/v1/[zip].json

to get information about a particular ZIP Code™.

This will return data to you in the following structure:

{
    "zip": string;
    "latitude": string;
    "longitude": string;
    "city": string;
    "state": string;
    "state2": string;
    "gridId": string;
    "gridX": number;
    "gridY": number;
}

Example

For the ZIP Code™ 99803, go to:

/api/v1/99803.json

and you will be returned the following information:

{
    "zip":"99803",
    "latitude":"58.301600",
    "longitude":"-134.419400",
    "city":"Juneau",
    "state":"Alaska",
    "state2":"AK",
    "gridId":"AJK",
    "gridX":194,
    "gridY":163
}
